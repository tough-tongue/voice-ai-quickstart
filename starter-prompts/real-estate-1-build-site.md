### What to Build

A single-page marketing site with smooth scroll. No login. No forms.

**Pages / Routes:**
- `/` — the main landing page
- `/slides` — channel partner slide deck index *(stub for now, link in navbar)*

**Landing page sections (top to bottom):**

1. **Navbar** — logo left, nav links right; transparent over hero, solid on scroll
2. **Hero** — full-viewport cinematic image, large headline, subtle scroll indicator
3. **Intro** — short editorial paragraph + property tagline, centered or left-aligned
4. **Highlights** — 3 key property stats (size, bedrooms, location) in a clean grid
5. **Gallery** — a row or grid of property images with a hover effect
6. **Footer** — logo, RERA number, copyright line

**Look & feel:**
- High-end, editorial, minimal — think luxury magazine, not a real estate portal
- Light background, dark text, one gold accent color
- Serif headings (e.g. Cormorant Garamond), clean sans-serif body (e.g. Figtree or Inter)
- Smooth scroll between sections
- Subtle fade-in animations as sections enter the viewport

**Images:** Placeholder images are fine — use descriptive `alt` text so they're easy to swap.
