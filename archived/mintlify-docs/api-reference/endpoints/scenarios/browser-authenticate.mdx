---
title: "Authenticate Browser"
api: "POST /v2/scenarios/browser/authenticate"
description: "Create a browser session for authenticated browser-use demos"
---

Creates a Browserbase context and session for scenarios that have the browser
tool enabled. Returns an `embed_url` the caller opens once to authenticate —
the auth state persists for subsequent browser actions in the same session.

Requires the scenario to have the browser tool configured.

<RequestExample>
```json
{
  "scenario_id": "693960de75f58a7b74c835bf"
}
```
</RequestExample>

<ResponseExample>
```json 200
{
  "embed_url": "https://www.browserbase.com/sessions/abc123/authenticate",
  "session_id": "browserbase_session_abc123",
  "context_id": "browserbase_context_xyz"
}
```
</ResponseExample>

<Note>
Open the `embed_url` in a browser once to complete authentication. The
auth state is stored in the Browserbase context and reused for the duration
of the scenario session.
</Note>
