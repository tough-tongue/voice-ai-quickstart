---
title: "Generate Advanced Scenario"
api: "POST /v2/scenario/generate-advanced"
description: "AI-generate scenario content from a name or context document"
---

Generate a complete scenario definition — persona, instructions, objectives, evaluation
rubric — using AI. Provide a name, a context document, or both.

## Request Parameters

<ParamField body="name" type="string">
  Short name or topic for the scenario (e.g. `"Sales cold call — SaaS product"`)
</ParamField>

<ParamField body="context" type="string">
  Background document: product brief, job description, company description, etc.
  The more context provided, the more specific the generated scenario.
</ParamField>

<ParamField body="prompt" type="string">
  Additional freeform instruction to guide generation
  (e.g. `"Make the prospect skeptical and price-sensitive"`)
</ParamField>

<ParamField body="reasoning_power" type="string" default="standard">
  Controls generation quality and speed. One of `fast`, `standard`, `deep`.
</ParamField>

<ParamField body="include_name" type="boolean" default="true">
  Whether to set the generated `name` field on the scenario
</ParamField>

<RequestExample>
```json
{
  "name": "Enterprise sales discovery call",
  "context": "We sell a B2B SaaS HR platform. Prospect is a 500-person company currently using spreadsheets.",
  "reasoning_power": "standard"
}
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": "693960de75f58a7b74c835bf",
  "name": "Enterprise sales discovery call",
  "ai_instructions": "You are Jordan, a skeptical HR Director at a mid-size company...",
  "voice": "alloy",
  "model": "gpt-4o"
}
```
</ResponseExample>
