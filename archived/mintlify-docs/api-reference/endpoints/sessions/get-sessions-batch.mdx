---
title: "Get Sessions Batch"
api: "GET /v2/sessions/batch"
description: "Fetch multiple specific sessions by ID in a single request"
---

Retrieve a list of sessions by their IDs. Much faster than calling
[Get Session](/api-reference/endpoints/sessions/get-session) in a loop
or filtering [List Sessions](/api-reference/endpoints/sessions/list-sessions-v2).

<ParamField query="session_ids" type="string" required>
  Comma-separated list of session IDs to fetch
  (e.g. `session_abc,session_def,session_xyz`)
</ParamField>
